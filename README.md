# Blacksmith Formula

Install the **Blacksmith Standard Edition** CLI:
```bash
$ brew tap nunchistudio/blacksmith-cli
$ brew install blacksmith-standard
```

Install the **Blacksmith Enterprise Edition** CLI:
```bash
$ brew tap nunchistudio/blacksmith-cli
$ brew install blacksmith-enterprise
```
